# Dignite Cms

Dignite Cms是一款基于[Abp Framework](https://github.com/abpframework)的CMS解决方案。它允许开发者在线定义字段，以满足各种前端页面内容展示的需求。此外，Dignite Cms还支持多租户、多站点、多语言等特性，为用户提供了高度灵活和全面的功能。
