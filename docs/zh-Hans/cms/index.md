# Dignite Cms

Dignite Cms是一个基于[Abp Framework](https://github.com/abpframework)开发的CMS模块，开发者通过自定义字段，满足前端页面数据展示的需求。

Dignite Cms支持多租户、多语言、页面模板、页面路由等特性，为开发者提供了灵活的定制开发功能。
